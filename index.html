Script to play, manipulate and download YouTube videos

Right click on the link below and "Copy link address". If you have no such option, the code is written below the link, select and copy it manually. Bookmark any page whose icon you like.
Edit the created bookmark's Properties / Location, and paste over its URL-address, that is, replace the bookmark's original location with this snippet of javascript code. Give it an adequate name / description you want. Finally, save the bookmark(let).

Also, you can just drag and drop the link to the Bookmarks Toolbar, or right click on it and select "Bookmark this link". It will have no icon attached to it this way.

On any YouTube video page, this bookmarklet will call an external player and load some extra functions which can be used with YouTube's native player as well. It doesn't work with live streams yet. To use it, you have to click on it on your bookmarks list each time you enter a video page, unless you have the code already running. Use the HTML5 emblem to switch between the native and external player. The program is able to

- restrict on-site advertisement,
- resize, zoom, rotate and move around video picture arbitrarily,
- change video playback rate ("speed" or tempo) from 0 or 25% to up to 500 or 800%,
- pick your favoured (and supported) video/audio codec and resolution for playback,
- download the video/audio content,
- download subtitles/closed captions (CC) (on bookmarklet line, edit the value of 'add_subs' (2) (comma separated list of ISO 639-1 languge codes) to add Google-Translated tracks of your choice),
- bypass age verification (experimental, native player only, without download links)

The script should at least work with newer versions of Firefox (24+). It should work now on most browsers, even older ones. Bugs are gonna stay forever...

Note: Users of FF-addon NoScript

  1. Take a look at the Settings of NoScript and allow (i.e. uncheck) Embedded Objects:

    <AUDIO> / <VIDEO>, <IFRAME>

  2. Add to NoScript Settings / Whitelist

    githubusercontent.com - program code is stored on this site.
    rawgit.com - #1 external proxy to load the main script.
    githack.com - #2
    allow-any-origin.appspot.com - #1 external proxy to get the video signature decryption data.
    cors-anywhere.herokuapp.com - #2
    crossorigin.me - #3

  3. (DEPRECATED) Look at Firefox's internal page "about:config" (type it on the address bar) and edit the following entry

    noscript.inclusionTypeChecking.exceptions

    - add:

    https://raw.githubusercontent.com/snarly/yt6/master/yt6.js

    If the player still won't show up, check the browser's "Webdeveloper's Console" to see which files NoScript shows having trouble with on a repeated try and add their full URL-address to this entry.
